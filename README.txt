Active G-Code		- contains the FreeCAD files, already sliced and ready to print on a Prusa I3 Mk3.
FreeCAD Files 		- contains the CAD files for the model.  They were produced using FreeCAD.
Project Archives	- contains archived models, either due to updates, failed prints, or redesigns.
STL Files       	- contains the export of a FreeCAD model, which is the input to a slicer.
